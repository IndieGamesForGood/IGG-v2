Setting Up Environment
----------------------

    $ easy_install virtualenv
    $ virtualenv --no-site-packages env
    $ source ./env/bin/activate
    $ pip install -r requirements.txt

Running Development Server
--------------------------

    $ source ./env/bin/activate
    $ ./runserver
